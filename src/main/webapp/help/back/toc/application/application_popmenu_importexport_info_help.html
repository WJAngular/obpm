当前页面描述：
  • 导入导出，对软件，模块，模块元素进行迁移。
功能说明
  • 导出：

    第一步：点击导出数据

    第二步：选择导出的类型，有软件，模块，模块元素三种类型

    软件：整个软件应用

    模块：相应模块中所有配置

    模块元素：对应模块中表单，视图，流程，报表以及交叉报表中的相应选择的对像

    注意：运行导出时，IE会有下载风险提示,右击提示选择下载文件后会返回到主页页面，再次进入模块后导入导出就可以成功了。

    第三步：选择保存路径，并生成以“年月日时分秒”命名规则的ZIP格式的压缩文件（注意：导入时是无需解压的）。

  • 导入：

    第一步：点击导入数据

    第二步：选择导入的类型，有软件，模块，模块元素三种类型，对应导出的类型选择。

    软件：整个软件应用

    模块：相应模块中所有配置

    模块元素：对应模块中表单，视图，流程，报表以及交叉报表中的相应选择的对像。

    第三步：选择导入对象（导出软件时生成的ZIP文件）的路径。
