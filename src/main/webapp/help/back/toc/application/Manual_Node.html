• 点击节点然后放到编辑页面
    在编辑页面双击节点（或选中再点击编辑按钮）进入节点编辑页面
    基本信息
    名称：节点的名称
    状态求签：节点状态
    审批进程中的表单绑定：
    点击按钮进入表单选择页面
    选择要绑定的表单
    可以设置表单字段：
    只读：字段对当前审批人只读
    修改：当前审批人可以修改字段
    隐藏：字段对对当前审批人隐藏
    ：勾上则对应的列全部勾上
    ：保存更改过表单设置，选择一个选中表单
    ：退出选择页面
    ：退出选择页面
    人员设置
    ：根据角色决定节点审批人
    点击按钮选择角色
    ：保存选中角色
    ：退出选择
    ：清除已选角色
    ：根据脚本决定节点审批人
    ：可以选择具体用户审批节点
    审批设置
    审批送出设置
    当下步有多个审批节点时，可以选择多个节点：
    如图（前台运行效果）：
    当下步有多个审批节点时，只能选择一个节点
    审批到达设置
    当上步有多个审批节点时，所有节点完成，流程可以到达
    当上步有多个审批节点时，只要任一节点完成，则流程到达
    操作设置
    回退设置：
    允许回退：
    默认回退：可以回退至前面的任一节点
    定制回退：可以回退至指定节点
    点击进入节点选择页面
    ：保存选中节点
    ：退出选择
    ：清除已选节点
    不允许回退：节点不允许回退
    回撤设置：
    设计：
    可回撤：当前审批人提交流程之后还可以回撤回来
    不可回撤：当前审批人提交流程之后就不能回撤了
    脚本：根据编写的脚本回撤
    条件类型：
    任一审批人通过，则节点通过：
    当流程有多个审批人的时候，只要其中一个审批通过，则节点通过
    所有审批人通过，则节点通过：
    当流程有多个审批人的时候，所有审批人通过，则节点通过
    通知提醒
    发送方式：
    邮件：以邮件方式发送
    手机短信：以手机短信方式发送
    站内短信：以站内短信发送
    通知类型：
    发送：发送的时候通知
    接收者：选择要发送给的接收者
    发送模板：选择要发送的模板
    到达：到达的时候通知
    发送模板：选择要发送的模板
    开启短信审批：是否开启短信审批
    回退：回退的时候通知
    回退类型：选择要回退的类型
    发送模板：选择要发送的模板
    过期：过期的时候通知
    是否通知上级：选择是否通知上级
    时限：选择时限
    单位：选择单位
    发送模板：选择要发送的模板
