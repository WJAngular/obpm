当前任务描述
  • 报表基本信息的维护页面。交叉报表是属于基本的报表，是行、列方向都有分组的报表。它能配置报表标题、是否进行行列汇总、报表的查询语句。
功能说明
  • 名称：设置交叉报表名称
  • 标题：报表标题，用于视图显示中的报表名称
  • 备注：对报表描述说明
  • 数据源：报表数据的来源试图
  • 显示行汇总：选择是否对报表进行行汇总，汇总方式有：汇总、平均
  • 显示列汇总：选择是否对报表进行列汇总，汇总方式有：汇总、平均
  • 签入：解除锁定，并确认其他用户登录后台能对当前交叉报表进行编辑
  • 签出：锁定当前编辑交叉报表内容，并确认其他用户登录后台无法对当前交叉报表进行编辑保存
  • 保存：保存当前页面信息
  • 退出：返回交叉报表列表界面
