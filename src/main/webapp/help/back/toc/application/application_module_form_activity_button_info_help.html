当前页面描述
  • 操作按钮的维护页面。“操作”是平台中“表单”和“视图”的动作，在页面以按钮的形式表现。可以为“表单”和"视图"页面分别配置保存、流程处理和删除、Exel导出等类型的操作按钮。
功能说明
  • 名称：设置表单操作按钮名称
  • 多语言标签：设置需要多语言显示的内容在前台可以看到多种语言显示
  • 类型：设置表单操作按钮类型
  • 图标：操作按钮在前台显示的图标
  • 状态：当前表单的流程状态与所设置的状态一致时，此操作按钮可见，否则按钮隐藏
  • 动作执行前脚本：系统在执行按钮的功能前运行“动作执行前脚本”
  • 动作执行后脚本：系统在执行按钮的功能后运行“动作执行后脚本”
  • 隐藏脚本：当系统运行“隐藏脚本”后结果返回true时则操作按钮隐藏，结果返回为false时按钮不隐藏
  • 只读脚本：当系统运行“只读脚本”后结果返回true时则操作按钮为只读状态，结果返回为false时按钮为正常状态
  • 转发：按钮类型为转发时候，在前台点击可以以邮件的形式转发
  • 保存并新建：保存当前页面信息，并跳转至新建操作按钮界面
  • 确定：确定当前操作并返回至操作列表界面
  • 取消：取消当前操作并返回至操作列表界面
