在编辑页面双击节点（或选中再点击编辑按钮）进入节点编辑页面
名称：节点的名称
状态标签：节点的状态
是否跨流程：
如果为“是”，则必须在“子流程名称”中选择子流程表单，当子流程启动时会同步新建子流程表单的文档，文档中的各文档项需根据“子流程名称”的脚本进行设置
子流程名称：
点击按钮进入流程列表
：退出选择
：清除已选流程
子流程表单名称：
点击按钮进入表单列表
：退出选择
：清除已选表单
子流程启动时调用的脚本：
子流程启动脚本，当流程流转到子流程节点，在启动子流程时所调用的脚本，即主流程切换到子流程时调用的脚本。
子流程返回主流程时调用的脚本：
子流程回调脚本，当子流程流程到结束节点，切换回主流程时所调用的脚本，即子流程切换到主流程时调用的脚本。
