当前任务描述
  • 关联节点信息的维护页面。流程节点是通过路径连接在一起的。
功能说明
  • 路径名称：节点显示名称
  • 表单：略
  • 字段：略
  • 比较：略
  • 操作：由表单字段、比较字符或手动输入的字符组成的脚本
  • 路径进入条件脚本：判断是否符合进入本路径的条件，当脚本返回值为true时为符合，false时为不符合。不符合的路径将不能被提交通过。
  • 路径执行脚本：路径通过时，需要执行的脚本。通常可以在此实现：调用发送邮件、短信提醒或与有关系统的数据接口等。
  • 路径送出检验脚本：判断是否能通过本路径将流程当前处理提交到下一个节点。当脚本返回一个非空的字符串时。校验不能通过，并将字符串提示给用户。当返回值为空时，校验通过，并成功提交下一个节点
  • 保存：略
  • 退出：略
