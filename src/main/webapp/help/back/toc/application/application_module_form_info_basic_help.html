当前任务描述
  • 表单基本信息维护页面。可对表单基本配置信息进行维护。
功能说明
  • 名称：表单的名称，此项为必填项，不能重名
  • 类型：分为普通，查询表单，普通（映射）表单

    普通，常规的表单配置

    查询表单，主要为视图列表配置查询头，要与视图配置结合使用

    映射表单，可以调用应用软件在同一个数据库中不属于平台生成的数据表，通过字段的映射，对该表进行数据查询，修改，添加，删除操作

  • 样式库：通过调用软件/库/样式库可以定义表单的设计样式
  • 显示操作历史日志：显示表单被修改过的记录，具体示例可以查看“软件缺陷管理系统模板”
  • 打开前执行脚本：打开表单前将会先执行的动作，例如：判断相应的用户才可进入该表单编辑，否者无法进入表单
  • 描述：略
