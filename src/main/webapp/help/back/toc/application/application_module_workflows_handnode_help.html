当前任务描述
  • 人工处理节点信息的维护页面。由指定的人员或角色完成当前环节的处理操作。
功能说明
  • 名称：节点显示名称
  • 状态标签：节点状态
  • 审批过程中的表单绑定：通过设置与当前节点绑定的表单，以及对表单的权限操作，可以实现对当前节点操作表单的权限控制。比如：某个字段只读、隐藏或可编辑等
  • 审批编辑模式：设计：通过设置审批组别，决定当前环节处理人员。代码：通过编写审批脚本，并根据脚本返回的结果决定当前环境处理人员。指定审批人：由上一环节处理人员在提交时，直接指定当前环节处理人员（在通过设计或代码确定人员范围内指定）
  • 审批组别：设置审批成员
  • 审批送出设置：略
  • 审批到达设置：略
  • 回退设置：有“允许回退”和“不允许回退”。允许回退  默认回退：可以回退至前面的任一节点。定制回退：可以回退至指定节点。不允许回退：该节点不能回退
  • 回撤设置：有“设计”和“脚本”。设计有“可回”跟“不可回撤”。 可回撤：当前审批人提交流程之后还可以回撤回来。不可回撤：当前审批人提交流程之后就不能回撤了
  • 条件类型：任一审批人通过，则节点通过：当流程有多个审批人的时候，只要其中一个审批通过，则节点通过。 所有审批人通过，则节点通过：当流程有多个审批人的时候，所有审批人通过，则节点通过
  • 发送方式：邮件：以邮件方式发送。手机短信：以手机短信方式发送。站内短信：以站内短信发送
  • 通知类型：略
  • 保存：略
  • 退出：略
