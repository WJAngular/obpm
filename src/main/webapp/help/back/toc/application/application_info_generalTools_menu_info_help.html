当前任务描述
  • 菜单信息维护页面。通过菜单维护功能，可以自定义软件的菜单（最终用户可见）内容以及菜单的组织形式，层级结构。
功能说明
  • 名称：设置菜单名称
  • 链接：设置该菜单相应的链接
  • 多语言标签：设置标签内容在前台能用多种语言显示
  • 上级：该菜单的上级菜单
  • 参数：该菜单可设置参数
  • 排序号：对菜单进行排序，需要每个菜单都加了排序号才有效果
  • 菜单类型：PC端菜单在最终用户通过浏览器操作软件时生效，手机端菜单在最终用户通过手机智能客户端操作软件时生效
  • 图标：设置不同的图标在前台显示。
  • 打开链接：设置不同打开链接在前台显示。
  • 保存并新建：保存当前页面信息，并跳转至新建菜单界面
  • 保存：保存当前页面信息
  • 退出：略
