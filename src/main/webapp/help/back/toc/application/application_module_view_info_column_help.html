  • 视图列列表的维护页面
功能说明
  • 创建列：点击"创建"按钮使用"创建"方法，用户需要在右边填写列的基本属性

    列名称：显示的列名

    列宽：默认列宽为自动分配宽度，如手动设定则按照设定值设置列宽。比如：我们把列宽设置为：20%，表明该字段占整个表格列宽的20%

    类型：选择FIELD会出现表单和字段的选择；选择SCRIPT则可对值脚本进行设置

    表单：显示列所对应的表单

    字段：所选表单对应的字段

    值脚本：通过值脚本的设置可对列值进行设置

    流程回退标识：在流程回退时会有标识显示

    是否汇总：对列作汇总操作

    隐藏脚本：控制列的显示与隐藏；如输入true在前台这个字段就会被隐藏

  • 批量创建：点击批量创建就可以多同时大量的创建操作
  • 移除：通过勾选已经创建好的列钱的框然后点击移除按钮就能实现移除
