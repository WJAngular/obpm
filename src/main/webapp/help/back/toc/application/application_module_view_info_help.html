当前任务描述：
  • 维护视图信息的页面。视图是数据查询及数据查询结果展示的页面。
功能说明：
  • 名称：设置视图名称，根据用户自定的规则输入合法的名称
  • 描述：设置视图描述信息，用于对此视图的简单作用描述
  • 视图类型：视图类型分为：列表视图、日历视图、树形视图、甘特视图、地图视图
  • 样式库：可以通过下拉方式进行选择，但是必须的在软件的样式库中创建好样式，此处才会有供选择的下拉选项
  • 打开类型：打开类型有：普通、当前区域显示、弹出层显示、网格显示
  • 查询模版：通过下拉的方式选择查询的模块，但是必须先创建好查询表单，此处才会有供选择的下拉选项
  • 相关表单：设置视图需要关联的表单，排序字段会根据此设置列出表单字段；过滤模块下如果采用设计模式，视图过滤查询也是根据相关表单的设置来查询数据库
  • 排序字段：可以设置视图查询时根据某列进行排序
  • 显示记录总数：设置视图列表是否显示查询表中的记录总数
  • 刷新：设置视图列表被包含为明细列表时，创建记录后是否刷新主表页面信息
  • 分页：设置视图列表是否做分页查询，可设定一页显示多少条记录
  • 只读：设置视图列表记录是否允许点击进入查看或编辑页面
  • 签入：解除锁定，并确认其他用户登录后台能对当前视图进行编辑
  • 签出：锁定当前编辑视图内容，并确认其他用户登录后台无法对当前视图进行编辑保存
  • 创建菜单：点击创建菜单，就会弹出一个对话框让你进行新建菜单，然后保存
  • 预览：点击预览可以查看视图配置信息，配置出的视图预定效果显示页面
  • 保存：保存当前页面信息
  • 退出：返回视图列表界面
