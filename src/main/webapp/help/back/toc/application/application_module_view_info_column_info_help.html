当前页面描述：
  • 维护视图列信息页面。设置视图列表中每列的显示名称和显示值等。
功能说明：
  • 创建列：点击“创建”按钮使用“创建”方法，用户需要在右边填写列的基本属性

    列名称：显示的列名。(2)列宽：默认列宽为自动分配宽度，如手动设定则按照设定值设置列宽。比如：我们把列宽设置为：20%，表明该字段占整个表格列宽的20%

    多语言标签：设置需要多语言显示的列在前台多语言显示

    类型：选择FIELD会出现表单和字段的选择；选择SCRIPT则可对值脚本进行设置

    表单：显示列所对应的表单

    字段：所选表单对应的字段

    是否为排序字段：设置字段在前台是否可以排序(升序、降序)

    流程回退标识：在流程回退时会有标识显示。

    是否汇总：对列作汇总操作

    显示方式：数据显示的方式(所有、部分)

    是否汇总：是否对数据进行汇总

    是否可见：列在前台是否可以看见，默认可见

    隐藏脚本：控制列的显示与隐藏；如输入true在前台这个字段就会被隐藏

  • 批量创建 ：点击批量创建就可以多同时大量的创建操作
  • 移除 ：通过勾选已经创建好的列钱的框然后点击移除按钮就能实现移除
