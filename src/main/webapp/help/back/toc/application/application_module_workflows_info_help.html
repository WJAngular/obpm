当前任务描述：
  • 流程信息的维护页面。流程是工作过程和环节的描述，流程由工作的多个步骤组成，每一步由指定的经办人填写指定的表单控件，谁都不能跨越权限，体现了一种制度和规范。 当前页面可配置流程的各种环节，和环节与环节之间的关联关系，而手工环节可以指定经办人，以及设置经办人所具有表单控件的权限，还可以对经办人进行自动催办。
功能说明：
  • 主题：流程的名称
  • 作者：默认为当前登录用户
  • 开始节点：流程的起点，同一个流程允许有多个节点
  • 结束节点：流程的终点，同一个流程允许有多个终点
  • 手动节点：人工处理节点
  • 自动节点：自动跳转流程的节点
  • 子流程节点：通过流程调用另外的流程
  • 挂接节点：暂停流程
  • 取消节点：取消流程
  • 终止节点：终止流程
  • 关联：表示节点与节点之间的关联
  • 签入：解除锁定，并确认其他用户登录后台能对当前流程进行编辑
  • 签出：锁定当前编辑流程内容，并确认其他用户登录后台无法对当前流程进行编辑保存
  • 保存：保存当前页面信息
  • 退出：返回流程列表界面
  • 模式切换：选中"代码"单选框时可切换到代码编辑界面。在界面中勾选"修改"复选框可进行代码的编辑
  • 移除：移除流程编辑器中选择的元素
  • 放大：放大流程编辑器
  • 缩小：缩小流程编辑器
