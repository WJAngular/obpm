当前任务描述
  • Excel导入配置信息的维护页面。通过Excel导入配置，可以实现将Excel数据导入到指定表单中，并完成批量录入，通过此功能既可以实现单表的导入，也可实现主从表的导入。Excel导入配置是用于将现有的数据导入对应配置好的表单的功能。它能配置Excel导入配置模板和Excel导入配置图关系信息。
功能说明
  • 名称：设置Excel导入配置的名称
  • 模版上传：导入Excel数据的标准模板，提供给最终用户下载，并按照此模板的样式准备数据
  • 图表:以图表方式显示Excel配置

  • 代码:以代码方式显示Excel配置

  • 添加主工作表：对应与Excel的Sheet，不管是配置单表导入，还是配置主从表导入，均需设置
  • 添加子工作表：对应Excel的Sheet，仅在主从表时需要设置
  • 添加列：添加工作表的列
  • 关联：配置元素之间的关联，如出现在工作表和列直接，则表示列属于关联的工作表，如出现列与列直接，则表示主工作表和子工作表通过这两个列进行关联
  • 移除配置元素：移除已经添加的元素
  • 编辑配置元素：对已存在的元素进行修改
  • 保存：保存当前页面信息
  • 退出：返回Excel导入配置列表界面
