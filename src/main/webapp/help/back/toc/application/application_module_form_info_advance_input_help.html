当前页面描述
  • 单行文本输入框，用于输入字符数据，常用表单控件，不可换行，字符长度受限，如输入数据较多，请选用“多行文本框”。
功能说明
  • 名称:略
  • 文本框类型:设置当行文本框类型。文本—能填写任何数据，数字—只能填写数字。
  • 文本类型:设置单行文本框文本类型，普通（可编辑的字段），密码（以●号显示值），只读（不可编辑字段），隐藏（对控件进行隐藏）。
  • 焦点切换键:设置单行文本框的焦点切换键，tap/tap or enter
  • (只读)只显示文本:设置当行文本框只读只显示文本：是（只读情况下只显示文本内容），否（只读情况下除了显示文本文本边框也会有显示）
