• 编辑流程详细说明

    1.：给流程起个名称
    2.：默认是当前登录用户
    3.编辑按钮：选中一个“节点”点击进入节点编辑页面
    4.移除按钮：选中一个“节点”点击移除节点
    5.取消选择按钮：选中一个“节点”点击取消选中
    6.放大按钮：点击放大编辑页面
    7.缩小按钮：点击缩小编辑页面
    8.手动节点：
    9.关联节点：用来关联节点
    10.自动节点
    11.开始节点：流程的开始节点
    点击节点然后放到编辑页面
    在编辑页面双击节点（或选中再点击编辑按钮）进入节点编辑页面
    名称：节点名称
    状态求签：节点状态
    12.完成节点：完成流程
    点击节点然后放到编辑页面
    在编辑页面双击节点（或选中再点击编辑按钮）进入节点编辑页面
    名称：节点名称
    状态求签：节点状态
    13.挂起节点：挂起流程
    点击节点然后放到编辑页面
    在编辑页面双击节点（或选中再点击编辑按钮）进入节点编辑页面
    名称：节点名称
    状态求签：节点状态
    14.终止节点：终止流程
    点击节点然后放到编辑页面
    在编辑页面双击节点（或选中再点击编辑按钮）进入节点编辑页面
    名称：节点名称
    状态求签：节点状态
    15.取消节点：取消流程
    点击节点然后放到编辑页面
    在编辑页面双击节点（或选中再点击编辑按钮）进入节点编辑页面
    名称：节点名称
    状态求签：节点状态
    16.SubNode(子流程节点)：
    点击节点然后放到编辑页面
