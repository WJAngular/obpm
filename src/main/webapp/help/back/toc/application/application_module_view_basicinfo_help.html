当前任务描述：
  • 维护视图信息的页面。视图是数据查询及数据查询结果展示的页面。
功能说明：
  • 名称：设置视图名称，根据用户自定的规则输入合法的名称
  • 查询模版：通过下拉的方式选择查询的模块，但是必须先创建好查询表单，此处才会有供选择的下拉选项
  • 样式库：可以通过下拉方式进行选择，但是必须的在软件的样式库中创建好样式，此处才会有供选择的下拉选项
  • 打开类型：打开类型有：普通、当前区域显示、弹出层显示、网格显示
  • 分页：设置视图列表是否做分页查询，可设定一页显示多少条记录
  • 数据来源表单：视图数据的来源表单
  • 确认：略
  • 退出：略
