当前任务描述
  • 维护表单信息页面。表单是用于配置数据采集页面的格式与存储结构。它能配置页面元素与布局相关信息，能为表单页面配置各类型操作按钮。
功能说明
  • 基本
  • 格式
  • 操作
  • 签入：解除锁定，并确认其他用户登录后台能对当前表单进行编辑
  • 签出：锁定当前编辑表单内容，并确认其他用户登录后台无法对当前表单进行编辑保存
  • 表单预览：对当前表单进行预览
  • 创建菜单：添加菜单的快捷方式，菜单类型为新建一条当前表单的文档
  • 重置字段编号：通过简单的拷贝粘贴复制过来的表单元素，可能会由于元素的ID重复引发错误，通过本功能可以对元素的ID进行重置
  • 清除数据：清除表单所对应的数据记录（慎重使用）
  • 一键生成视图：生成视图的快捷方式，可方便的为表单生成相关的视图
  • 名称：新建表单的名称
  • ：选择表单的类型(1.普通2.普通(映射)3.查询表单4.模板表单)
  • 样式库：设置表单样式引用的样式库
  • 显示操作历史：设置是否显示对表单操作的历史
  • 是否可打开脚本：设置在前台视图中任选一条已存在记录是否可进入相应表单信息界面
  • 是否可编辑脚本：设置表单控件是否可以编辑
  • 描述：对表单的描述
  • 保存：保存当前页面信息
  • 退出：返回表单列表界面
