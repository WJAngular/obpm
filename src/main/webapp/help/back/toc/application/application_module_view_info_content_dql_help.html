当前任务描述：
  • 使用DQL模式的目的是让数据查询语句变得简洁。
功能说明：
  • 使用DQL模式的好处主要有以下两点：
  • (1)DQL模式能使字段名简化，操作起来比较方便。
  • (2)DQL模式能对传递过来的参数进行处理。
  • DQL语法介绍：
  • 语法格式：
  • $formname=’表单名’ [AND filter1……
  • 说明：
  • $formname：标识查询的表单；
  • AND filter1……：表示带过滤条件查询。
  • 使用：
  • var doc = getCurrentDocument();
  • var dql = “$formname=’表单1’ AND code=’TASK AND $id<>’”+doc.getId()+”’”;
  • var datas = queryByDQL(dql);
  • var rtn = “”;
  • if (datas.size()>0){
  • rtn = “系统中已经存在编号为：TASK0001 的记录！”;
  • }
  • rtn;
  • 总结：
  • DQL语句中使用到了code字段与$id字段进行过滤。其中$id表示使用系统字段ID；code字段为用户通过表单配置的字段名。在DQL语句中凡使用系统字段都需要在字段名前加”$”符号，用户配置字段可直接使用。
