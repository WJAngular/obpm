当前任务描述
  • 子流程信息的维护页面。子流程信息是流程信息的一部分。
功能说明
  • 名称：节点显示名称
  • 状态标签：节点状态
  • 是否跨表单：如果为“是”，则必须在“子流程表单名称”中选择子流程表单，当子流程启动时会同步新建子流程表单的文档，文档中的各文档项需根据“子流程表单名称”进行设置
  • 子流程名称：选择包含的子流程名称，可根据模块和名称查询
  • 子流程表单名称：选择包含的子流程表单名称，可根据模块和名称查询
  • 子流程启动时调用的脚本：子流程启动脚本，当流程流转到子流程节点，在启动子流程时所调用的脚本
  • 子流程返回主流程时调用的脚本：子流程回调脚本，当子流程流程到结束节点，切换回主流程时所调用的脚本
  • 保存：略
  • 退出：略
