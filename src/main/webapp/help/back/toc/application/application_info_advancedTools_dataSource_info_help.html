当前任务描述
  • 数据源信息的维护页面。数据源是用于记录软件连接数据库的具体信息。它能配置连接数据库类型、连接URL、驱动、用户名、密码等信息。
功能说明
  • 名称：设置数据源名称
  • 数据库类型：选择系统连接的数据库
  • 数据驱动类：选择数据类型之后自带不需要手动输入
  • 地址：设置链接数据库的字符串
  • 用户名称：登录数据库的用户名
  • 密码：登录数据库的密码
  • 连接池大小:设置数据库的连接池大小

  • 连接超时时间:设置数据连接超时的时间

  • 测试数据库：点击"测试数据库"，会弹出成功与否的提示
  • 保存：保存当前页面信息
  • 退出：返回数据源列表界面
