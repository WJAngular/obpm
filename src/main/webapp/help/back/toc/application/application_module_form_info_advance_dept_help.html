功能说明

  • 刷新:详情
  • 重计算:详情
  • 手机：详情
  • 部门等级:
  • 0:指企业域本身；
  • 1:所创建的顶级部门；
  • 2:顶级部门下的下属部门。
  • 上级部门:当有多个部门选择框时会读取其他部门选择框字段名称，并对此进行上级设置
  • 受用户限制:只会显示当前登录用户所属部门
  • 默认选择类型:
  • 用户所属部门:默认值为用户默认所属的那个部门；
  • 第一个选项:下拉框的第一选项值。
  • 空选项:下拉框的第一个值为空值
