当前任务描述
  • 任务信息的维护页面。任务可以让系统在特定日期特定时间执行指定的任务内容，并通过终止条件结束任务。
功能说明
  • 任务名称：设置任务名称

  • 创建人:系统自动获取创建人，不可更改

  • 类型：（1）导出数据：任务会自动的对软件进行导出备份；（2）脚本：任务可以用脚本来设置任务内容
  • 启动类型：设置任务启动类型
  • 重复方式：（1）立刻：任务会马上执行，不重复（2）每日：任务会每天执行一次（3）每周：任务会每个星期执行一次（4）每月：任务会每月执行一次
  • 修改时间:

    最后修改的时间
  • 描述:对任务内容的描述

  • 任务内容：设置任务定时执行的内容（脚本）
  • 终止条件脚本：用脚本设置任务终止条件，返回值：true 任务停止 | false 任务继续执行
  • 运行时间:设置任务运行的具体时间

  • 保存：保存当前页面信息
  • 退出：返回任务列表界面
