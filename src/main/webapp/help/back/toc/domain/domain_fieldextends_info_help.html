当前任务描述
  • 扩展字段信息的维护页面。用于扩展部门和用户的基础信息字段
功能说明
  • 显示标签：设置当前扩展字段在所属模块列表中的列头显示名称
  • 排序：设置当前字段在同一所属模块中的所有扩展字段（field1到field10）的范围内，在所属列表中的显示的位置，其数值为0以上，数值越小在列表中就越靠前
  • 所属字段：设置当前所属扩展字段
  • 类型：设置当前扩展字段的类型（日期或字符串类型）
  • 是否在列表中显示：设置当前扩展字段在相应模块的列表中是否显示
  • 所属模块：设置当前扩展字段的所属模块，用户或部门
  • 保存：保存当前页面信息
  • 保存并新建：保存当前页面信息，并跳转到新建扩展字段界面
  • 退出：返回扩展字段列表界面
