当前任务描述
  • 用户信息的维护页面
功能说明
  • 名称：用户姓名
  • 帐号：用户登录系统时所用到的账号
  • 密码：用户登录系统时所用到的密码
  • 电子信箱：接收通知的邮箱
  • 手机：接收短信通知的手机号码
  • 日历：当前用户的工作日历
  • 状态：若为激活则可正常登录，否则无法登录
  • 上级：当前用户的上级用户
  • 代理人用户：允许代理人协助进行业务处理
  • 备注：设置用户描述说明
  • 代理开始日期：允许代理人协助进行业务处理的开始时间
  • 代理结束日期：代理人协助进行业务处理的结束时间
  • 设置企业域管理员权限：设置企业域管理员
  • 即时通信设置：允许即时通信设置
  • 部门：当前用户所属部门（可同时隶属于多个部门）
  • 角色：当前部门所属角色（可同时设置为多个角色）
  • 保存并新建：保存当前页面信息，并跳转至新建用户界面
  • 保存：保存当前页面信息
  • 退出：返回用户列表界面
