功能描述
1：文件管理：文件管理为后台管理人员管理文件夹及文件或者将文件夹及文件分配权限给指定用户去管理
2：共享根目录：显示当前选择的文件夹目录路径
3：搜索：输入关键字可模糊搜索文件名
4：分页功能
5：选择文件夹下分页显示文件列表
  • "图标"：文件类型以图标形式来显示
  • "名称"：文件名
  • "大小"：文件大小
  • "上传日期"：文件上传日期
  • "操作"：该文件具有操作功能
  • "查看"：点击查看可查看该文件内容
  • "删除"：点击删除将删除该文件
  • "下载"：点击下载可下载该文件到本地
  • "共享文件"：点击共享文件链接到权限页面，选择一个权限，点击确定即可完成共享文件
  • "取消共享"：点击取消共享该文件将取消共享
6：文件夹树：以树形的形式列出所有文件夹
