返回

getLastCount2(headText, isYear, isMonth, isDay, digit)

概述

返回“前缀 +年月日+ 计数器的当前计数值”。

返回值

String

参数

1、headTextString

计数器名称，同时作为前缀的字符串

2、isYearboolean

前缀中是否包含年份

3、isMonthboolean

前缀中是否包含月份

4、isDayboolean

前缀中是否包含日期

5、digitint

整数型，指定计数器序列号的位数，如长度不足在计数值前面补0

示例

描述:

实现getLastCount2(headText, isYear, isMonth, isDay, digit)函数的应用，自动编号的最后编号



iScript 代码:


//表单某字段值为计数器自动编号的最后编号 (值脚本)
(function(){
var prefix = "QJDTEST";
var val = getItemValue("商品编号");
if(val != null && val.trim().length() <= 0){ 
   var code = getLastCount(prefix);
   if(code<=0){
      val=countNext2(prefix,true,false,false,4); 
   }else{
      val =getLastCount2(prefix,true,false,false,4); //自动编号的最后编号
   }
}
return val;
})()
