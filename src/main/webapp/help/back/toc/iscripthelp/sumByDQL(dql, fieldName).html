返回

sumByDQL(dql, fieldName)

概述

根据dql汇总符合条件的Document的指定字段。

返回值

double

参数

1、dqlString

查询符合条件语句

2、fieldNameString

文档的字段名

示例

描述:

实现sumByDQL(dql, fieldName)函数的应用，根据dql汇总符合条件的Document的指定字段(DQL)



iScript 代码:


//两个表单A、B，B表单某字段值为A表单某字段值总和 (值脚本)
(function(){
var dql = "$formname  = '字段汇总A表单DQL'";
var value = sumByDQL(dql, "A单价");
return value;
})()
