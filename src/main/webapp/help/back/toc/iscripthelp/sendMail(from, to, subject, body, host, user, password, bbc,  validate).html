示例

描述:

实现sendMail(from, to, subject, body, host, user, password, bbc, validate)函数的应用，发送邮件给指定用户



iScript 代码:


//在表单界面点击某个按钮，想给表单中指定的用户发送邮件，可以在类型为“无”的动作执行脚本中写代码
(function(){
var from=getItemValueAsString("发送人地址");
var to=getItemValueAsString("接收人地址");
var subject=getItemValueAsString("主题");
var content=getItemValueAsString("内容");
var host=getItemValueAsString("邮件服务器地址");
var user=getItemValueAsString("邮件服务器用户名");
var pwd=getItemValueAsString("密码");
var bbc=getItemValueAsString("秘密抄送地址");
var validate=true;
sendMail(from, to, subject, content, host, user, pwd, bbc, validate);
})()
注意：该方法发送成功没有信息提示
