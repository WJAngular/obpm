返回

getViewProcess()

概述

创建视图业务对象。

返回值

cn.myapps.core.dynaform.view.ejb.ViewProcess

参数

示例

描述:

实现getViewProcess()函数的应用，创建视图业务对象。



iScript 代码:


//通过创建视图业务对象，然后根据名称获取视图对象
(function(){
var process = getViewProcess(); //视图业务对象
var applicationId=getApplication();
var view= process.getViewByName("表单控件测试单行文本框",applicationId);
println(view);
return view;
})()
