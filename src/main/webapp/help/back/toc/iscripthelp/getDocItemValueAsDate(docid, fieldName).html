返回

getDocItemValueAsDate(docid, fieldName)

概述

获取文档中Item的值,且以日期形式返回。

返回值

java.util.Date

参数

1、docidString

文档标识ID

2、fieldNameString

文档的字段名

示例

描述:

实现getDocItemValueAsDate(docid, fieldName)函数的应用，获取指定文档的Item值



iScript 代码:


//表单某字段值为指定文档的指定字段的值(值脚本)
(function(){
var doc =getCurrentDocument();
var docid = doc.getId();
var value=getDocItemValueAsDate(docid, "文档日期");
return value;
})()
