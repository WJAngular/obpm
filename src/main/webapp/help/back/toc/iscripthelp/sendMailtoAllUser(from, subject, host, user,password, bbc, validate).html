sendMailtoAllUser(from, subject, content,host, account,pwd, bbc, validate)

概述

发送邮件提醒给所有用户。

返回值

无

参数

1、fromString

发送人地址

2、subjectString

发送人地址

3、contentString

内容

4、hostString

邮件SMTP服务器地址

5、accountString

邮件服务器用户名

6、pwdString

用户密码

7、bccString

秘密抄送地址

8、validateboolean

是否校验

示例

描述:

实现sendMailtoAllUser(from, subject, content,host, account,pwd, bbc, validate)函数的应用， 可通过此方法发送邮件给该企业域下有填写Email地址并且激活的所有用户



iScript 代码:


//在表单界面点击某个按钮，想给当前企业域下所有的用户发送邮件，可以在类型为“无”的动作执行脚本中写代码
(function(){
var from=getItemValueAsString("发送人地址");
var subject=getItemValueAsString("主题");
var content=getItemValueAsString("内容");
var host=getItemValueAsString("邮件服务器地址");
var account=getItemValueAsString("邮件服务器用户名");
var pwd=getItemValueAsString("密码");
var bbc=getItemValueAsString("秘密抄送地址");
//or-------
var from="**@teemlink.com";
var subject="**";
var content="**";
var host="teemlink.com";
var account="**";
var pwd="******";
var bbc="********@qq.com";
var validate=true;
sendMailtoAllUser(from, subject, content,host, account,pwd, bbc, validate)
})()
