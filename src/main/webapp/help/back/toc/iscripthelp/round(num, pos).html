返回

round(num, pos)

概述

提供精确的小数位四舍五入处理。

返回值

double

参数

1、numnumber

需要四舍五入的数字

2、posint

小数点后保留几位

示例

描述:

实现round(num, pos)函数的应用，四舍五入函数



iScript 代码:


//表单中有两个字段A、B，B字段的值为A字段四舍五入后的值 (值脚本) 
(function(){
var value=getItemValueAsDouble("价格");
var rtn="";
if(value!=null){
   rtn=round(value,2);
}
return rtn;
})()
