返回

getRelateDocument()

概述

获取关联的文档，当包含元素不存在父子关系时生效。

返回值

cn.myapps.core.dynaform.document.ejb.Document

参数

1、无

示例

描述:

实现getRelateDocument()函数的应用，获取关联的文档，当包含元素不存在父子关系时生效(在视图内容中使用DQL)



iScript 代码:


//子关联文档的数据根据主关联文档的字段值进行过滤
(function(){
var dql = "$formname='fm_price' "; //子关联文档的表名
var rdoc =getRelateDocument();
if(rdoc!=null){
var linkmanName = rdoc.getItemValueAsString("商机名"); //主关联文档的字段
dql += " and 商机名='"+ linkmanName +"'";
dql+=" and (商机名 <> null and 商机名 <> '')" ;
}
return dql;
})()
