返回

getItemValueAsDouble(fieldName)

概述

获取当前打开文档中Item的值

返回值

Double

参数

1、fieldNameString

当前打开文档的字段名

示例

描述:

实现getItemValueAsDouble(fieldName)函数的应用，获取当前文档Item值



iScript 代码:


//判断表单某字段值小于0时，默认值就为1
(function(){
var value= getItemValueAsDouble("金额");
println("金额："+value+"----------");
if (value<0){
    value=1;
}
return value;
})()
