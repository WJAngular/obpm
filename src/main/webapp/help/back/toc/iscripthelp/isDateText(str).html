返回

isDateText(str)

概述

检查其参数是否为日期格式的字符串。是，返回true；否，则返回false。

返回值

boolean

参数

1、strString

字符串型参数

示例

描述:

实现isDateText(num)函数的应用，日期字串校验(写在控件只读脚本中)



iScript 代码:


//表单中有两个字段A、B，B字段是否为只读是根据A字段值是否为日期字符串类型 (只读脚本) 
// A字段值为日期字符串时，B控件是只读的
(function(){
var value=getItemValueAsString("时间");
var rtn=false;
if(value!=null && value.trim().length()>0){
   if(isDateText(value)){
       rtn=true;
    }
}
return rtn;
})()
