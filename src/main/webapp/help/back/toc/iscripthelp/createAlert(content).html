返回

createAlert(content)

概述

创建警告对话框。

返回值

无

参数

1、contentString

提示内容。如：“单据填写有误无法保存！”

示例

描述:

实现createAlert(content)函数的应用，创建警告对话框



iScript 代码:


//表单保存单据前，对字段进行校验 (保存动作执行前脚本)
(function(){
var name = getItemValueAsString("姓名");
if (name!=null && name.trim().length()<=0){
    createAlert("姓名字段未填写，不能保存，请输入！");
}
})()
