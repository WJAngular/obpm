返回

getDocItemValueAsDouble(docid, fieldName)

概述

获取文档中Item的值,且以double形式返回。

返回值

double

参数

1、docidString

文档的id标识

2、fieldNameString

文档的字段名

示例

描述:

实现getDocItemValueAsDouble(docid, fieldName)函数的应用，获取指定文档的Item值



iScript 代码:


//表单某字段值为指定文档的指定字段的值(值脚本)
(function(){
var doc =getCurrentDocument();
var docid = doc.getId();
var value=getDocItemValueAsDouble(docid, "文档金额");
return value;
})()
