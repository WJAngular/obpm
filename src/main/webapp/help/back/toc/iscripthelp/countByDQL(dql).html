返回

countByDQL(dql)

概述

根据dql统计符合条件的Document数量。

返回值

long

参数

1、dqlString

查询符合条件语句

示例

描述:

实现countByDQL(dql)函数的应用，根据域名查询文档集合（DQL）



iScript 代码:


//两个表单A、B，B表单某字段值为查询出A表单的文档个数 (值脚本)
(function(){
var dql = "$formname='统计文档个数DQL'";
var t_count = countByDQL(dql);
var rtn =0;
if(t_count > 0) {
   rtn = t_count;
}
return rtn;
})()
