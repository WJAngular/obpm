返回

parseInt(text)

概述

将数字文本转换成整型并返回。

返回值

int

参数

1、text

数字文本参数

示例

描述:

实现parseInt(text)函数的应用，将数字文本转换成整型并返回



iScript 代码:


//表单有A、B两个字段，B字段值为A字段转换成整型后的值(值脚本)
(function(){
var value=getItemValueAsString("价格");
var rtn=0;
if(value!=null && value.trim().length()>0){
   rtn=parseInt(value);
}
return rtn;
})()
