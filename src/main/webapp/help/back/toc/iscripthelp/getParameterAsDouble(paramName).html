返回

getParameterAsDouble(paramName)

概述

获取参数值,以浮点型的形式返回

返回值

String

参数

1、paramNameString

参数名

示例

描述:

实现getParameterAsDouble(paramName)函数的应用，获取参数值,以浮点型的形式返回



iScript 代码:


//表单中某字段值为参数表中某参数值以浮点型式返回
(function(){
var params=getParamsTable();
params.setParameter("p1",'10.25');  //设置参数
var value = getItemValueAsString("获取参数");
if (value==null || value.trim().length()<=0){
value = getParameterAsDouble ("p1");  //获取参数值
}
return value;
})()
