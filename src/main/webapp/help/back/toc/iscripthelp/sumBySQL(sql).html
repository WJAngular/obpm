返回

sumBySQL(sql)

概述

根据sql汇总符合条件的Document指定字段。

返回值

double

参数

1、sqlString

汇总符合条件语句

示例

描述:

实现sumBySQL(sql)函数的应用，根据sql汇总符合条件的Document指定字段(SQL)



iScript 代码:


//两个表单A、B，B表单某字段值为A表单某字段值总和 (值脚本)
(function(){
var sql = "select sum(item_A单价) from tlk_字段汇总A表单SQL";
var value = sumBySQL(sql, "A单价");
return value;
})()
