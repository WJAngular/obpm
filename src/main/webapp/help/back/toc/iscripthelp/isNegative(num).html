isNegative(num)

概述

判断是否为负数。

返回值

boolean

参数

1、numnumber

数字型参数

示例

描述:

实现isNegative(num)函数的应用，负数校验



iScript 代码:


//两个表单A、B，B表单某字段值为查询出A表单的文档个数 (值脚本)
(function(){
var value=getItemValueAsString("编号");
var rtn=false;
if(value!=null && value.trim().length()>0){
   if(isNegative(value)){
      rtn=true;
   }
}
return rtn;
})()
