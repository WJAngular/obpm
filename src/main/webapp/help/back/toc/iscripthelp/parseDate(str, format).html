返回

parseDate(str, format)

概述

将字符串按给定格式转换为日期型。

返回值

java.util.Date

参数

1、strString

待转换的字符串,需符合format指定的格式

2、formatString

日期格式

示例

描述:

实现parseDate (str,fromat)函数的应用，日期转换函数



iScript 代码:


//表单某字段值为转换格式之后的日期 (值脚本)
(function(){
var starttime = getItemValue("字符串");
var startdate=null;
if(starttime!=null && starttime.trim().length()>0){
    startdate= parseDate(starttime,"yyyy-MM-dd HH:mm:ss"); 
}
return startdate;
})()
