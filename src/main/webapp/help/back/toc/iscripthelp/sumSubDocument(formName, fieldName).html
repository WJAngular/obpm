返回

sumSubDocument(formName, fieldName)

概述

根据子文档名和字段名，获取当前打开文档的子文档中字段的值总和。

返回值

double

参数

1、formNameString

当前打开文档的子文档名

2、fieldNameString

子文档的字段名

示例

描述:

实现sumSubDocument(formName, fieldName)函数的应用，根据子文档名和字段名，获取当前打开文档的子文档中字段的值总和



iScript 代码:


//表单为主子表单，主表单某控件值为当前文档子文档某个字段值的总和（值脚本）
(function(){
var formName="被汇总字段值的子文档B";
var fieldName="金额";
var value=sumSubDocument(formName, fieldName);
return value;
})()
