返回

toFloor(num)

概述

返回小于等于其数字参数的最大整数。

返回值

int

参数

1、numnumber

数字参数

示例

描述:

实现toFloor(num)函数的应用，取最大整数



iScript 代码:


//表单中有两个字段A、B，B字段的值为A字段值的最大整数 (值脚本)
(function(){
var value=getItemValueAsDouble("价格");
var rtn="";
if(value!=null){
   rtn= toFloor (value);   //获取value的最大整数
}
return rtn;
})()
