返回

createObject(pathText)

概述

生成系统类对象。

返回值

Object

参数

1、pathTextString

类的名字空间，如"cn.myapps.util.DateUtil"

示例

描述:

实现createObject(pathText)函数的应用，创建指定类对象(写在下拉框选项脚本中)



iScript 代码:


(function(){
var opts = createObject("cn.myapps.core.dynaform.form.ejb.Options"); 
opts.add("A","A");
opts.add("B","B");
opts.add("C","C");
return opts;
})()
